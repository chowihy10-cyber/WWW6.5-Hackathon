# 🌌 MemoMuseum (赛博星轨)

## 📖 项目简介
MemoMuseum 是一个基于 Web3 的空间记忆与去中心化契约验证平台。在这里，个人记忆汇聚成人类的星轨。用户可以将情感、高光时刻刻印在 3D 星海中，或者发起带有质押性质的“军团契约”挑战，通过社区的“见证广场”进行社交验证（Social Verification）。

## ✨ 核心功能
1. **深空微光频段 (3D 空间记忆)**：基于地理位置算法，将用户的文字转化为全屏交互式 3D 星空上的微光节点。
2. **状态驱动的契约挑战**：用户发起挑战并质押 MTK 代币，到期后需提交自证进入 3 天的“全宇宙公示期”。
3. **去中心化陪审团 (见证广场)**：任何人均可通过星轨探测器寻找需要见证的灵魂，投出“达成”或“证据不足”的选票。
4. **情感化粒子反馈**：交互自带极具仪式感的粒子特效（成功时的金色星粉、失败时的蓝色流星）。
5. **记忆跨维度导出**：支持将链上数据一键导出为“温馨手工书”格式的 PDF。

## 🛠️ 技术栈
* **前端**：React, Vite, TailwindCSS
* **Web3 交互**：Ethers.js v6
* **视觉渲染**：Globe.gl (3D地球), @tsparticles (粒子系统)
* **智能合约**：Solidity
* **部署网络**：Avalanche Fuji Testnet
